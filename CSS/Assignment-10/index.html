The min() function

The min will select the smallest value from the list of comma-separated list of values. I have used 50% and 300px.

Resize the browser to see the effect

The max() function

The max will select the highest value from the list of comma-separated list of values. I have used 50% and 300px.

Resize the browser to see the effect

The clamp() function

The clamp() takes in three parameters, repectively,minimum value, preferred value and maximum value. It clamps a middle value within the range of values between a defined minimum bound and maximum bound.
